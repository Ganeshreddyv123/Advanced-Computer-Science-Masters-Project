# Advanced-Computer-Science-Masters-Project
Heart Disease Risk Prediction
A machine learning project that predicts the likelihood of heart disease in patients using clinical data. The project compares multiple ML models to find the most accurate one and examines how feature selection affects model performance and explainability.
Dataset

Source: Kaggle – Heart Disease UCI Dataset
14 features: age, sex, chest pain type, resting blood pressure, cholesterol, fasting blood sugar, ECG results, max heart rate, exercise-induced angina, ST depression, fluoroscopy vessel count, and Thallium stress test results
Target variable: presence or absence of heart disease (binary)
No personally identifiable information included

Models Used

Logistic Regression – simple, interpretable baseline model
Random Forest – ensemble method that captures non-linear relationships
XGBoost – gradient boosting for high accuracy and efficiency

Evaluation Metrics
Models are compared using Accuracy, Precision, Recall, and F1-Score to determine the best performer.
Tools & Libraries
Python, Jupyter Notebook, pandas, numpy, scikit-learn, xgboost, matplotlib, seaborn
How to Run

Clone the repo
Run pip install -r requirements.txt
Open Ganesh_Heart_Disease_Prediction.ipynb and run all cells
